# Project Orientation

## Folder Structure
- /Sources/ - Swift source code (TerminalNotifierApp)
- /Tests/ - Test files (TerminalNotifierTests)
- /Resources/ - App resources (Info.plist, entitlements)
- /scripts/ - Build automation (build-bundle.sh)
- /docs/ - Documentation and reference materials
- /archive/ - Archived original Objective-C and Ruby code

## Rules
- Always place files in their designated folders
- Never commit broken code
- Run `swift build` before committing
- Document significant changes
